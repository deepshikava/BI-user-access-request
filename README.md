# BI-user-access-request

It's Single Page Web Application used by Business Insights - BI team for granting & revoking certain tool access for some specific BI users.

Used React JS, Bootstrap, Redux for state management, performed CRUD operation for grant & Revoke access using REST API's.

**Get Started:**

Initially run `npm install` to install all supporting dependencies and then run `npm start` to start the app.
